# Mock Credit Card APIs
The Credit Card APIs are mock REST APIs designed using Mocky.io. All APIs have random response data generated by faker.js in Mocky.io.

## Base URLs

The base URL of the APIs are as follows:
`https://6a32b6dac6ca2aee43857dc6.mockapi.io/credit-card-customers-api/`

## Authentication

These APIs are publicaly available mock APIs. So authentication is not required to access these APIs.

## Credit Card Endpoints


### Create Credit Card

- [**POST** `v1/credit-card-customer-details`](create.md)

### Update a card

- [**PUT** `v1/credit-card-customer-details/:id`](update.md)

### Retrieve a card

- [**GET** `v1/credit-card-customer-details`](retrieve.md)

### List all cards

- [**GET** `v1/credit-card-customer-details`](retrieve-all.md)

### Delete a card

- [**DELETE** `v1//credit-card-customer-details/:id`](delete.md)
